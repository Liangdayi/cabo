# Cabo游戏卡牌交互动画 - 验证清单

- [x] 检查1: 抽牌动画是否实现
  - 从抽牌堆抽牌时，卡片有飞出动画
  - 从弃牌堆捡牌时，卡片有飞出动画

- [x] 检查2: 弃牌动画是否实现
  - 弃牌时，卡片有飞入弃牌堆的动画
  - 动画流畅自然

- [x] 检查3: 交换动画是否实现
  - 交换卡片时，新卡片有移动到目标位置的动画
  - 旧卡片有移动到弃牌堆的动画
  - 动画顺序正确，清晰展示交换过程

- [x] 检查4: 技能动画是否实现
  - 发动技能时，有相应的卡片交互动画
  - 不同技能类型有不同的动画效果

- [x] 检查5: 动画性能是否良好
  - 动画流畅不卡顿
  - 不影响游戏的响应速度
  - 在不同设备上表现一致

- [x] 检查6: 动画与游戏逻辑是否同步
  - 动画不影响游戏的正常流程
  - 动画完成后游戏逻辑正确执行

- [x] 检查7: 代码结构是否清晰
  - 动画代码结构清晰，易于维护
  - 与现有代码风格一致

- [x] 检查8: 所有动画场景是否覆盖
  - 抽牌场景
  - 弃牌场景
  - 交换场景
  - 技能发动场景
