# Cabo卡波微信小程序游戏

## 项目简介

Cabo是一款2-4人的数字卡牌策略游戏，玩家需要通过抽牌、交换、弃牌等行动，尽量让自己面前4张牌的总和尽可能小。当有玩家总分达到100分时游戏结束，总分最低的玩家获胜。

## 游戏规则

### 一、游戏准备
- **目标**：让面前4张牌总和尽量小，总分先到100分游戏结束，最低分获胜
- **卡牌**：0-13数字牌共52张
- **开局**：每人4张暗牌，只准偷看2张；中央牌堆+1张弃牌堆，随机起始玩家

### 二、回合行动（三选一）
1. **抽牌堆**：抽1张看后，可直接弃掉（7-12可发动能力）或交换
2. **捡弃牌堆**：拿最上1张，必须用来交换
3. **喊CABO**：跳过行动，其他人各再玩一轮，本轮结束

### 三、交换规则
- **单张交换**：抽1换1，抽牌堆来的暗置，弃牌堆来的明置
- **多张交换**：1张换面前多张同数字牌，验证成功则替换，失败则新牌入队，换3张以上失败还要多抽1张惩罚

### 四、特殊能力（7-12弃牌时可用）
- **7/8**：偷看自己1张暗牌
- **9/10**：间谍看别人1张暗牌
- **11/12**：交换自己与他人1张牌，正反面不变

### 五、一轮结束
- 有人喊CABO + 所有人多一轮 → 结束
- 牌堆抽空 → 直接结束
- 未到100分就重洗开局，最低分玩家起始

### 六、计分与胜利
- 喊CABO且全场最低 → 0分
- 喊CABO但有人更低 → 总和+10分
- 其他人：按自己总和计分
- 总分≥100分游戏结束，总分最低赢
- 平局看最后一轮得分，更低者胜
- 特殊：总分刚好100分，可重置为50分（整局1次）

## 核心功能

1. **房间系统**：创建房间、加入房间、设置玩家人数
2. **游戏逻辑**：发牌、抽牌、弃牌、交换牌、特殊能力
3. **CABO机制**：喊CABO后其他玩家额外回合
4. **计分系统**：计算得分、判断胜负
5. **多人对战**：支持2-4人游戏

## 技术实现

- **前端**：微信小程序原生开发
- **状态管理**：小程序页面数据管理
- **游戏引擎**：纯JavaScript实现游戏逻辑
- **UI设计**：响应式布局，深色主题

## 如何使用

1. **开发环境**：微信开发者工具
2. **导入项目**：在微信开发者工具中导入项目目录
3. **运行项目**：点击运行按钮，在模拟器或真机上测试
4. **创建房间**：进入首页，选择玩家人数，点击"创建房间"
5. **加入房间**：进入首页，输入房间号，点击"加入房间"
6. **开始游戏**：所有玩家准备后，房主点击"开始游戏"

## 注意事项

- 本项目为模拟实现，实际多人对战需要WebSocket后端支持
- 特殊能力的完整实现需要更多的UI交互
- 卡片交换的完整实现需要选择目标卡片的弹窗
