# Cabo 卡牌游戏

## 项目介绍

Cabo是一款策略性卡牌游戏，玩家需要通过记忆和策略来最小化自己的手牌分数。游戏的核心乐趣在于未知性——玩家不知道自己当前手牌的总和，需要通过记忆和推理来做出最优决策。

本项目是一个微信小程序实现，支持单人对战AI玩家，完整的实现了Cabo游戏的所有核心功能。（当前V1.0版本仅支持单人对战AI玩家，后续V2.0版本将加入 multiplayer 功能）

### 技术栈
- **前端框架**: 微信小程序原生开发
- **游戏引擎**: 自研状态机引擎
- **AI系统**: 基于规则的智能AI玩家
- **测试框架**: Jest单元测试

## 游戏规则

### 基本规则
1. **游戏人数**: 2-4人
2. **牌组**: 标准52张扑克牌（不含大小王）
3. **初始手牌**: 每人4张，暗置摆放
4. **获胜条件**: 游戏结束时总分最低者获胜

### 卡牌分数
| 卡牌 | 分数 |
|------|------|
| A | 1分 |
| 2-10 | 面值分数 |
| J | 11分 |
| Q | 12分 |
| K | 13分 |
|仅花色|0分|

### 游戏流程

#### 1. 开局偷看
- 游戏开始时，每位玩家选择2张暗牌进行查看
- 查看时间3秒，之后牌暗置回原位
- 玩家需要记住这些牌的位置和内容

#### 2. 回合操作
玩家在自己的回合可以选择：
- **抽牌堆抽牌**: 从牌堆顶部抽一张牌
- **弃牌堆捡牌**: 从弃牌堆顶部捡一张牌

抽牌/捡牌后可以：
- **交换**: 用抽到的牌换自己的一张或多张牌
- **发动技能**: 如果是7-12牌，可以发动特殊技能
- **弃牌**: 直接丢弃抽到的牌

#### 3. 特殊技能
| 牌 | 技能名称 | 效果 |
|------|----------|------|
| 7/8 | 偷看自己 | 查看自己的一张暗牌 |
| 9/10 | 间谍 | 查看对手的一张暗牌 |
| 11/12 | 交换 | 与对手交换一张牌 |

**注意**: 发动技能后，该牌进入弃牌堆并标记为"不可捡取"

#### 4. 喊CABO
- 玩家可以在自己的回合喊"CABO"
- 喊CABO后，其他玩家各进行一次回合，然后回合结束
- **成功**: 手牌分数最低，本轮得0分
- **失败**: 手牌分数不是最低，本轮得分 = 手牌分数 + 10分惩罚

#### 5. 回合结束
当以下情况发生时，回合结束：
- 有玩家喊CABO，且其他玩家各完成一次回合
- 牌堆抽空

回合结束后：
- 显示每位玩家的手牌分数
- 计算本轮得分
- 如果有玩家总分达到100分，游戏结束

#### 6. 新回合开始
- 重新发牌
- 执行开局偷看环节
- 继续游戏

### 计分规则
| 情况 | 本轮得分 |
|------|----------|
| 喊CABO成功 | 0分 |
| 喊CABO失败 | 手牌分数 + 10分 |
| 未喊CABO | 手牌分数 |

**特殊规则**: 如果玩家总分正好达到100分，可以重置为50分（每局游戏只能触发一次）

- 历史记录管理
- 分数统计功能

## 玩法示例

### 示例1：基础回合
```
1. 玩家A从牌堆抽到一张7
2. 玩家A选择"发动技能"→ 选择偷看自己的第2张牌
3. 系统显示第2张牌是红桃K（0分）
4. 7进入弃牌堆，标记为不可捡取
5. 回合结束，轮到玩家B
```

### 示例2：喊CABO
```
1. 玩家A记得自己的牌是：K(0分)、A(1分)、2(2分)、3(3分) = 6分
2. 玩家A认为自己的分数最低，喊"CABO"
3. 其他玩家各进行一次回合
4. 回合结束，计算分数：
   - 玩家A: 6分 → CABO成功 → 本轮0分
   - 玩家B: 15分 → 本轮15分
   - 玩家C: 8分 → 本轮8分
```

### 示例3：多张交换
```
1. 玩家A从牌堆抽到一张5
2. 玩家A选择"交换"→ 选择自己的两张5进行交换
3. 验证成功：手牌中有两张5
4. 抽到的5替换两张手牌5，两张手牌5进入弃牌堆
5. 玩家A的手牌减少到3张
```

### 示例4：技能连锁
```
1. 玩家A从牌堆抽到一张9（间谍技能）
2. 玩家A选择"发动技能"
3. 选择对手B，查看B的第3张牌
4. 发现B的第3张牌是红桃K（0分）
5. 玩家A记住这个信息，用于后续决策
```

## 快速开始

### 环境要求
- 微信开发者工具
- Node.js (用于运行测试)

### 安装步骤
1. 克隆项目到本地
2. 使用微信开发者工具打开 `cabo-game` 目录
3. 点击"编译"运行项目

### 运行测试
```bash
npm install
npm test
```
